# 看板与资产

> 更新日期：2026-07-21

计划跑完后，用右侧看板检查并微调结果。本章说明常用 Tab 与资产使用注意点。

## 主要 Tab

| Tab | 做什么 |
|-----|--------|
| **「剧本详情」** | 查看 / 调整剧本标题与正文类信息 |
| **「角色」** | 人物资产；可 **「＋ 新建角色」**，生成或更换形象 |
| **「空镜」** | 场景类资产；可 **「＋ 新建空镜」** |
| **「分镜」** | 镜头列表（胶片条或表格）；点开单镜改旁白、字幕、画面挂接等 |
| **「剪辑」** | 时间轴预览与进入精修（见下一章） |

分镜视图可在胶片 / 表格等形态间切换（界面提供 **「分镜视图切换」**）。

## 分镜里常改什么

打开某一镜后，常见可调内容包括：

- 旁白、对白与字幕时段
- 关联的 **角色** / **空镜**（画面参考）
- 画面与视频时段（故事书多为静图；AI 视频模式还会涉及视频片段）
- 单镜或所选资产的 **「重新生成」**

改完以界面保存 / 写回为准；有引用的资产删除会受限，界面会提示引用关系。

## 资产复用（概念即可）

- 人物、场景（空镜）、道具等可在项目内复用，降低系列片重复劳动。
- 文字设定与生成出的图片 / 音频是分开管理的：改文案不一定自动重画，需要时可显式重新生成。
- 跨剧本共享以产品界面能力为准；不确定时先在当前剧本内复用。

## 与对话的配合

- 大范围改结构、重跑某阶段：回左侧对话说明意图，让编排再规划。
- 局部微调文案或单镜画面：优先在看板直接改，再按需重新生成媒体。
