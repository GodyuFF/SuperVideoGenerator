# AI 配置

> 更新日期：2026-07-22

本章说明如何在 **「AI 配置」** 中配好各服务，使对话与生成可用。Key 由你自备，默认只保存在本机。

## 打开入口

- 顶栏 **「AI 配置」**
- 未配置时工作台提示 **「去配置 AI」**

配置完成后点 **「保存并返回对话」**（或 **「保存配置」**）。

## 各 Tab 做什么

| Tab | 作用 | 首次建议 |
|-----|------|----------|
| **「LLM」** | 对话与多 Agent 编排 | 必填：服务商、模型、API Key；启用 LLM ReAct，状态应为「LLM 已配置」 |
| **「图片」** | AI 生图（角色、空镜、分镜画面等） | 保持启用，填写生图服务商 Key |
| **「TTS」** | 旁白 / 对白配音 | 默认 **edge** 即可，一般无需 Key |
| **「视频」** | AI 生视频（文生 / 图生等） | 仅故事书时可关闭；用 AI 视频或图生视频前须启用并填写视频 API Key |
| **「剪辑导出 / FFmpeg」** | 成片合成所用 FFmpeg | 默认用内置即可；仅在本机路径特殊时再改 |
| **「Embedding / RAG」** | 共享池 `list` 语义检索（辅助 Agent 发现可复用资产） | 进阶；首次可跳过 |

页面上可对图片、TTS 等做「生成测试图」「生成试听」等探测，便于确认 Key 有效。

## TTS 旁白与角色音色

- **旁白默认音色**（TTS Tab 的默认 voice）：作用于分镜里 `character_ref` 为空的旁白 clip。可按片种选男声或女声，不必沿用系统预填。
- **角色对白**：角色资产的 `gender`（男/女）与 `tts_voice` 决定对白音色；系统会按性别匹配，男角色不会回落到女声默认旁白。
- **呼吸空隙**：合成时会为每镜保留句前 / 句间 / 镜尾静音，镜时长按「说话 + 空隙」对齐，画面可长于纯朗读，避免全程不停说话。

## 本机保存位置

| 方式 | 说明 |
|------|------|
| 应用内保存 | 写入本机 `data/ai_config.json`（不入 Git） |
| 环境变量 | 仓库根目录 `.env`（可参考 `.env.example`），与应用内配置效果等价 |

请勿把含真实 Key 的文件提交到版本库。更多「勿提交」路径见 [常见问题](07-faq.md)。

## 常见卡点

- 工作台一直提示配置 AI → 检查 LLM Tab 是否已保存且状态为已配置。
- 故事书能出片但 AI 视频失败 → 到 **「视频」** Tab 启用并填写对应服务商 Key。
- 换电脑后配置丢失 → `data/` 与 `.env` 默认不随安装包迁移，需重新填写或自行备份。
