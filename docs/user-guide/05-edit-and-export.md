# 剪辑与导出

> 更新日期：2026-07-21

本章说明如何在 **「剪辑」** Tab 预览时间轴，进入 **「剪辑修改」** 精修，并导出成片。

## 剪辑 Tab

计划中的剪辑规划完成后：

1. 打开右侧 **「剪辑」** Tab。
2. 预览时间轴上的画面、旁白与字幕编排。
3. 需要精修时，点击 **「剪辑修改」** 进入剪辑助手（Edit Studio）。

分镜详情里若出现「前往剪辑 Tab」类入口，作用相同：跳到全片时间轴视图。

## 剪辑修改（专业剪辑）

在剪辑助手中可以：

- 多轨调整片段位置、裁切与层级
- 预览成片效果
- 使用顶栏 **「导出」** 输出成片

界面能力基于集成的剪辑组件；日常使用按预览 → 调整 → 导出即可，无需了解内部工程名。

## 导出成片

1. 在剪辑助手顶栏打开 **「导出」**。
2. 按提示选择分辨率、是否含音频等参数。
3. 导出完成后下载或保存 MP4（桌面端可能直接在资源管理器中定位文件）。

说明：

- 成片导出以剪辑助手内顶栏 **「导出」** 为准。
- 若提示媒体尚未加载完成，等待素材加载成功后再导出。
- 故事书成片依赖本机 FFmpeg（默认内置；路径可在 AI 配置的 **「剪辑导出 / FFmpeg」** 调整）。

## 导出后还想改？

- 小改字幕 / 镜头：回看板或剪辑助手继续改，再重新导出。
- 大改剧情：回对话说明需求，必要时新建剧本以免与已锁定风格冲突。
