# 剪辑与导出

> 更新日期：2026-07-22

本章说明如何在 **「剪辑」** Tab 预览时间轴，进入 **「剪辑助手」** 精修，并导出成片。

## 剪辑 Tab

计划中的剪辑规划完成后：

1. 打开右侧 **「剪辑」** Tab。
2. 预览时间轴上的画面、旁白与字幕编排。
3. 需要精修时，点击 **「剪辑修改」** 进入剪辑助手。

分镜详情里若出现「前往剪辑 Tab」类入口，作用相同：跳到全片时间轴视图。

## 剪辑助手（专业剪辑）

点击 **「剪辑修改」** 后进入全屏 **「剪辑助手」**：左侧素材库、中央预览监视器、右侧属性面板、底部多轨时间轴。

![图6 剪辑助手](assets/figure-06-edit-studio.png)

*图6 剪辑助手（素材库 · 预览 · 属性 · 多轨时间轴）*

### 界面分区

| 区域 | 作用 |
|------|------|
| **顶栏** | 主题 / 语言、**「导出」**、**「完成并关闭」**、更多选项 |
| **左侧素材** | 媒体 / 音频 / 文字等素材；可切换列表或网格，并可 **「导入」** |
| **中央预览** | 当前帧与字幕叠加；下方为时码、播放、适应画布、全屏 |
| **右侧属性** | 选中时间轴片段后，编辑位置、大小、音量等；未选中时为空态提示 |
| **底部时间轴** | 文字 / 视频 / 音频等多轨；支持撤销重做、裁切、删除、磁性时间轴与缩放 |

### 常用操作

在剪辑助手中可以：

- 在时间轴上拖动、裁切片段，调整文字 / 画面 / 配音对齐
- 点击预览区播放，核对字幕与旁白是否同步
- 选中片段后，在右侧属性面板微调参数
- 使用顶栏 **「导出」** 输出成片
- 完成后点 **「完成并关闭」** 返回工作台（改动会写回当前剧本时间轴）

日常使用按 **预览 → 调整 → 导出** 即可，无需了解内部工程细节。

## 导出成片

1. 在剪辑助手顶栏打开 **「导出」**。
2. 按提示选择分辨率、是否含音频等参数。
3. 导出完成后下载或保存 MP4（桌面端可能直接在资源管理器中定位文件）。

说明：

- 成片导出以剪辑助手内顶栏 **「导出」** 为准。
- 若提示媒体尚未加载完成，等待素材加载成功后再导出。
- 故事书成片依赖本机 FFmpeg（默认内置；路径可在 AI 配置的 **「剪辑导出 / FFmpeg」** 调整）。

## 导出后还想改？

- 小改字幕 / 镜头：回看板或剪辑助手继续改，再重新导出。
- 大改剧情：回对话说明需求，必要时新建剧本以免与已锁定风格冲突。
