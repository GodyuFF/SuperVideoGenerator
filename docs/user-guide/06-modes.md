# 视频风格与模式

> 更新日期：2026-07-21

本章对照对话区可选的 **视频风格**，以及各自对 AI 配置的要求。风格在剧本首次发送后锁定。

## 内置风格一览

| 视频风格 | 适合 | AI 配置要求 |
|----------|------|-------------|
| **「故事书模式」** | 首次上手；静图 + 旁白合成为片 | LLM + 图片 + TTS；**视频** Tab 可关闭 |
| **「AI 视频模式」** | 需要模型生成动态视频片段 | 另须在 **「视频」** Tab 启用 AI 视频并填写视频 API Key |
| **「画面图生视频」** | 以已有画面为参考做图生视频 | 同上，需要可用的视频生成能力与画面资产 |

自定义风格（若在提示词工作台中添加）显示名称以界面为准；未勾选 AI 生视频子模式时，行为接近故事书（图文合成）。

## 故事书（推荐入门）

1. 选 **「故事书模式」**，关闭 **目标模式**。
2. 按 [从零到成片](01-first-video.md) 走完对话 → 看板 → 导出。
3. 画面以静图为主，成片由剪辑时间轴合成。

## AI 视频模式

1. 在 **「AI 配置」→「视频」** 启用 AI 视频生成，填写服务商 Key（界面说明所支持的服务商）。
2. 新建剧本，视频风格选 **「AI 视频模式」**（须在首次发送前选好）。
3. 对话推进时，分镜侧会更多涉及视频片段与生成；可在分镜中查看成片模式、参考源等。

费用与耗时通常高于故事书；确认卡片里若出现费用预估，请按需确认。

## 画面图生视频

1. 同样需要视频 API 可用。
2. 风格选 **「画面图生视频」**。
3. 生成依赖剧本画面作为参考源；可在分镜相关面板中选择 / 调整画面参考（界面文案为 AI 生视频参考源等）。

## 切换风格的正确方式

- **同一剧本内**：首次发送后不可改风格。
- **需要另一种风格**：新建剧本，在发送前选择目标风格。

图片风格、预计时长等可选提示也会在首次发送时一并锁定（若已填写）。
