# 前端国际化（i18n）

> 更新日期：2026-07-09

SuperVideoGenerator 前端基于 **i18next + react-i18next**，支持 **简体中文（zh-CN，默认）** 与 **English（en）** 切换。语言选择持久化于 `localStorage` 键 `svg.locale`。

## 范围

**已双语化（用户可见 UI）：**

- SVF 主应用：路由页、顶栏、看板、对话、A2UI 表单、资产卡片、手动编辑、Edit Studio 外壳
- OpenCut 嵌入层完整子树：按钮、Tab、下拉/右键菜单、工具栏 tooltip、对话框、属性侧栏、素材面板、时间轴、预览、导出、Landing/Onboarding

**不在范围：**

- LLM 对话正文与用户项目/剧本名称
- API 下发的 A2UI 动态字段
- 音效/字幕转写语言列表本身（业务数据，非界面文案）

## 目录结构

```
apps/web/src/i18n/
├── config.ts              # initI18n、命名空间注册、LOCALE_STORAGE_KEY
├── LocaleProvider.tsx     # 包裹 App，监听语言变化
├── LocaleSwitcher.tsx     # 顶栏「中文 / EN」切换
├── useAppTranslation.ts   # SVF 页面 Hook
├── translateRegistryLabel.ts  # 注册表 labelKey → 文案
└── locales/
    ├── zh-CN/             # common, nav, board, chat, settings, editor
    │   └── opencut/       # timeline, assets, export, properties, shortcuts, dialogs, landing, common
    └── en/                # 同上

apps/web/src/editor/opencut/i18n/
└── useOpencutT.ts         # OpenCut 各 namespace 封装
```

## 开发约定

### SVF 组件

```tsx
import { useTranslation } from "react-i18next";

function MyPanel() {
  const { t } = useTranslation("board");
  return <button>{t("actions.save")}</button>;
}
```

跨 namespace：`t("common:actions.save")`。

### OpenCut 组件

```tsx
import { useOpencutT } from "@opencut/i18n/useOpencutT";

function MyToolbar() {
  const { tTimeline, tAssets } = useOpencutT();
  return <span>{tTimeline("split")}</span>;
}
```

### 注册表 labelKey

静态注册表（actions、properties、assets tabs 等）使用 `labelKey` 而非硬编码 `label`：

```ts
{ id: "media", labelKey: "opencutAssets.tabs.media" }
```

渲染时调用 `translateRegistryLabel(labelKey)` 或对应 `tAssets("tabs.media")`。

### 参数注册表（属性侧栏）

`params/registry.ts` 等注册表保留英文 `label` 作为 fallback；渲染层通过 `translateParamLabel(param.key)` / `translateParamOption(param.key, option.value)`（`editor/opencut/i18n/translateParamLabel.ts`）解析 `opencutParams` 命名空间。

格式：`{namespace}.{rest}`，namespace 为第一个 `.` 前的段（如 `opencutAssets` → `tAssets`）。

## 语言切换

- 组件：`LocaleSwitcher`（项目首页、工作台、剪辑器顶栏等）
- 持久化：`persistLocale()` / `readStoredLocale()`（`config.ts`）
- 初始化：`App.tsx` 内 `LocaleProvider` 包裹整棵 React 树

## 手动验收清单

1. 顶栏切换「中文 / EN」，主要页面按钮与 Tab 即时更新
2. 刷新页面后语言保持（检查 `localStorage.svg.locale`）
3. 进入 Edit Studio / OpenCut：时间轴工具栏、素材 Tab、右键菜单、导出、快捷键、属性侧栏均为当前语言
4. 无障碍：时间轴、预览、数值字段等 `aria-label` 随语言切换
